================================================================================
                              DATA STORAGE
================================================================================

All data stored in JSON format in data/ directory:

config.json:
  - Guild-specific settings
  - Log channel IDs
  - Jail role/channel
  - Roleplay enabled status

hardbans.json:
  - User ID → {reason, banned_by}
  - Persistent across restarts

fake_perms.json:
  - Guild → {users: {}, roles: {}}
  - Permission lists per user/role

roleplay_gifs.json:
  - Guild → {command: gif_url}
  - Custom GIF mappings

vc_data.json:
  - Channel ID → {owner_id, trusted[], blocked[], locked}
  - Voice channel state

antinuke_data.json:
  - Guild → {whitelist: {users[], roles[]}, thresholds: {}}
  - Anti-nuke configuration
